# CNN Image Classifier

This project demonstrates a basic implementation of a Convolutional Neural Network (CNN) for image classification using the MNIST dataset.

# Features
- Data preprocessing
- CNN model building
- Model training & evaluation
- Accuracy visualization

# Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

# Results
Achieved good accuracy on MNIST dataset with a simple CNN model.

## 📂 Project Structure
- notebooks/ → Jupyter Notebook
- src/ → Model scripts
- outputs/ → Graphs and results
